# Continual-Learning-Dataset
Two datasets for continual learning tasks, designed to support incremental and lifelong learning experiments.

This repository contains two datasets designed for **continual learning** tasks. These datasets are curated to support **incremental learning** and **lifelong learning** experiments, helping researchers develop models that can learn new tasks over time without forgetting the previously learned tasks.

The datasets are shared through Google Drive links for easy access and use in research.

## üóÇÔ∏è Datasets Overview

### 1. **exAAPD**
- **Description**: A dataset for continual learning based on the **AAPD** task. It consists of images and labels related to the **AAPD** classification task, designed to test a model's ability to incrementally learn new categories over time.
- **Format**: Image data (JPEG/PNG), labels in CSV format.
- **Link**: [Google Drive - exAAPD](<https://drive.google.com/drive/folders/1g9s_UiaTVC0GK80s56tiR-Tby-Jbi-wy?dmr=1&ec=wgc-drive-globalnav-goto>)

### 2. **exLitCovid**
- **Description**: This dataset contains text data related to the **LitCovid** dataset, used to evaluate the model's ability to adapt to different topics over time.
- **Format**: Text data (CSV/JSON).
- **Link**: [Google Drive - exLitCovid](<https://drive.google.com/drive/folders/1fKFqXa0KgTo8NOsiUtipkgoZfgJ2_AK3?dmr=1&ec=wgc-drive-globalnav-goto>)

### 3. **exMeSH**
- **Description**: A dataset related to **Medical Subject Headings (MeSH)**, designed for experiments involving hierarchical classification and task adaptation.
- **Format**: Structured text (CSV/JSON).
- **Link**: [Google Drive - exMeSH](<https://drive.google.com/drive/folders/1rzsEPQA4nHhU6u16ewMh-Fedjxf1BaPO?dmr=1&ec=wgc-drive-globalnav-goto>)

### 4. **exPFD**
- **Description**: The **PFD** dataset, focusing on learning with multiple, progressively complex tasks in **physical force detection**.
- **Format**: Image + sensor data.
- **Link**: [Google Drive - exPFD](<https://drive.google.com/drive/folders/1V1ocybV6PnMRCduPEhUReqvDzd_8xzIE?dmr=1&ec=wgc-drive-globalnav-goto>)
